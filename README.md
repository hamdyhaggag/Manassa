# Manassa E-Learning App

## Overview

Manassa is an E-Learning mobile application developed using Flutter, designed to provide users with a seamless learning experience. It offers a diverse range of courses and resources aimed at enhancing users' knowledge and skills.

## Features

- **Course Catalog**: Explore a wide range of available courses.
- **Interactive Learning**: Engage with interactive learning materials.
- **Progress Tracking**: Keep track of your progress within each course.
- **User Profiles**: Personalize your learning experience with user profiles.
- **Notifications**: Stay updated with notifications for new courses or updates.

## Installation

1. Clone the repository:

```bash
git clone https://github.com/yourusername/manassa-elearning.git
```

2. Navigate to the project directory:

```bash
cd manassa-elearning
```

3. Install dependencies:

```bash
flutter pub get
```

4. Run the app:

```bash
flutter run
```
